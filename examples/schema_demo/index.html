Schema composition

Before & after

Twelve node types and four marks, wired by hand — or composed in one expression.

Before — verbose Schema.new/1
Schema.new(%{
  "nodes" => [
    {"doc", %{"content" => "block+"}},
    {"paragraph", %{"content" => "inline*", "group" => "block"}},
    {"blockquote", %{"content" => "block+", …}},
    {"horizontal_rule", %{"group" => "block"}},
    {"heading", %{"content" => "inline*", …}},
    {"code_block", %{"content" => "text*", …}},
    {"text", %{"group" => "inline"}},
    {"image", %{"inline" => true, …}},
    {"hard_break", %{"inline" => true, …}},
    {"bullet_list", %{"content" => "list_item+", …}},
    {"ordered_list", %{"content" => "list_item+", …}},
    {"list_item", %{"content" => "paragraph block*", …}}
  ],
  "marks" => [
    {"link", %{…}},
    {"em", %{}},
    {"strong", %{}},
    {"code", %{"code" => true}}
  ]
})
After — Basic + List
alias ProsemirrorEx.Schema.{Basic, List}
alias ProsemirrorEx.Model.Schema

schema =
  Schema.new(%{
    "nodes" =>
      Basic.nodes()
      |> List.add_list_nodes(
           "paragraph block*",
           "block"
         ),
    "marks" => Basic.marks()
  })
